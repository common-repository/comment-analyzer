"Comment Analyzer" is a plugin that helps you moderate the comments on your WordPress site. It will analyze user sentiment i.e positive,negative or neutral feedback on your blog post.
 
== Description ==
"Comment Analyzer" adds a custom field(Sentiment Type) at edit comments section. After this plugin activation, whenever user comment on your post.
It will get moderate on the basis of open source third party library.  

== Installation == 
This section describes how to install the plugin and get it working.
1. Download and install the plugin from WordPress dashboard. You can also upload the entire “Comment Analyzer” folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Go to edit comments sections, and you will see new custom field(Sentiment Type).
